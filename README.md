# Student Management System (Console App)

A simple **C# console application** that manages student records.  
This project demonstrates the use of classes, lists, loops, switch statements, and relational operators.

---

## Features
- **Add New Student**  
  Prompt the user to enter student details (ID, Name, Grade).  
  Validates that the grade is between 0 and 100.

- **View All Students**  
  Displays all students in a **formatted table** showing:
  - ID
  - Name
  - Grade
  - Status (Pass/Fail based on grade >= 60)

- **Calculate Average Grade**  
  Computes and displays the average grade of all students.

- **Find Student by ID**  
  Searches for a student by their ID and displays their details.

- **Update Student Grade**  
  Allows updating the grade of a student by ID.

- **Delete Student**  
  Removes a student record by ID.

- **Display Statistics**  
  Shows the total number of students and the average grade.

- **Exit**  
  Ends the program.
